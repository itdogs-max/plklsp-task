## 學歷與早年經歷
- 中學：聖士提反女子中學。
- 大學：香港大學文學院卒業，之後曾到格拉斯哥大學及史丹福大學進修。

## 公職生涯重點
- 1996–1998 年：人民入境事務處處長，是香港史上首位出掌紀律部隊的女性部門首長。
- 1998–2003 年：香港特區政府保安局局長，也是首位女性保安局局長。
- 2008–2025 年：香港立法會議員，代表香港島及香港島西等選區，多屆連任。
- 2011 年：創立並出任新民黨主席，定位為偏向中產及建制派的政黨。
- 2022 年起：出任行政會議召集人，是香港回歸後首位擔任此職的女性。

## 政治形象與爭議
- 她在擔任保安局局長時，是《基本法》第 23 條立法的主要推動者，被形容立場強硬、鷹派。
- 2003 年因 50 萬人上街遊行反對 23 條草案，最終政府擱置立法，她以個人理由辭去保安局局長職務。
- 媒體曾以「具爭議的鐵娘子」形容她，批評者覺得她威權、強硬，支持者則視她為有魄力、果斷的女政治人物。
- 部分言論（例如對普選、對外傭、對皮草的看法）引起輿論爭議，常成為新聞焦點。

## 葉劉淑儀現有個人網站改善建議
- 首頁資訊偏多，新訪客不容易一眼看到「最重要的兩三句」介紹她是誰。
- 版面風格較傳統官式，個人品牌感（專屬色、強烈視覺主題）不太突出。
- 內容呈現方式較像新聞稿列表，故事感與個人敘事不強，對學生來說較難代入。

## 簡單的自我檢查
- 一打開頁面，有沒有清楚看到她是誰、在做什麼？
- 顏色、字體和間距看起來有一致感嗎？
- 我有沒有加一點自己的想法，而不是完全照抄範例？
